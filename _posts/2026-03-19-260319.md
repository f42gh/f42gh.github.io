## 始めてのWebサイト作成
GitHub Pages + Jekyll で自分のサイトを立ち上げた。

## 動機
ポートフォリオとアウトプットの為の場所がほしかった。学んだことを .md で書いて push するだけで公開される仕組みにした。

## 構成
ホスティング: GitHub Pages
静的サイトジェネレータ: Jekyll
デプロイ: main ブランチへの push → GitHub が自動ビルド・デプロイ

/_posts/ に YYYY-MM-DD-title.md 形式でファイルを置くと、Jekyll が HTML に変換してトップページの一覧に自動追加する。/_layouts/default.html で共通の <head> や CSS 読み込みを管理しているので、記事は本文だけ書けばいい。
